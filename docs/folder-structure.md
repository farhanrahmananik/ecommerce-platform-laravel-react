# Folder Structure

This scope prepares the project folder structure only. Feature implementation will happen in later scopes.

## Backend

The `backend` folder contains the Laravel API application. Its application folders provide dedicated locations for actions, contracts, data objects, enums, filters, API controllers, requests, resources, policies, services, and shared support code as the backend grows.

## Frontend

The `frontend` folder contains the React and Vite application. Its source folders provide dedicated locations for application setup, assets, reusable components, context, feature modules, hooks, layouts, pages, routing, API services, styles, utilities, and constants as the frontend grows.

## Current Scope

Only the folder skeleton is included at this stage. No authentication, database, product, cart, or order logic has been added.
